Module 2
Data Imputation
Apply and compare three imputation strategies on missing values. Select a technique to see its effect on the feature distribution before and after.
MODE IMPUTATION
Replace missing categoricals with the most frequent value. Fast, simple. Best when missingness is low and MCAR.
KNN IMPUTATION
Use k=5 nearest neighbours to infer the missing value. Preserves relationships between features. Slower but more accurate.
MICE / ITERATIVE
Multiple Imputation by Chained Equations. Models each feature with missing data as a function of others. Gold standard for MAR data.
