Combines early and late fusion: first fuse Sources A+B at the feature level (early), train separate models on the fused set and Source C independently, then combine their outputs (late). Captures both intra-source and inter-source patterns.

TECHNIQUE 05 · ATTENTION FUSION
Attention Fusion — Adaptive
A learned attention mechanism dynamically assigns weights to each source per-sample based on feature context. Unlike fixed weighted fusion, attention weights vary for each row — high-capital-gain records may weight Source C (financial) more heavily.
◈HOW ATTENTION FUSION WORKS
1
Each source passes through its own encoder
h_A = encoder_A(X_A)   h_B = encoder_B(X_B)   h_C = encoder_C(X_C)   # shape: (N, d)
2
Attention scores computed per sample
e_i = W_attn @ concat(h_A, h_B, h_C) + b   # (N, 3) unnormalised scores
3
Softmax normalises to per-sample weights
alpha = softmax(e_i, axis=1)   # alpha shape: (N, 3), sum=1 per row
4
Weighted sum of source representations
h_fused = sum(alpha[:,i] * h_i for i in [A,B,C])   y = classifier(h_fused)
◈ATTENTION WEIGHTS BY INCOME GROUP
High-earners (>50K) rely more on BLS wage data (Source B). Low-earners on demographic features (Source A).

TECHNIQUE 07 · BAYESIAN FUSION
Bayesian Fusion — Probabilistic
Uses Bayes' theorem to combine evidence from multiple sources. Each source updates a shared posterior belief about the class label. Mathematically principled — handles uncertainty and conflicting source evidence gracefully.
◈BAYESIAN UPDATE RULE
P(Y=1 | X_A, X_B, X_C) ∝ P(X_A | Y=1) × P(X_B | Y=1) × P(X_C | Y=1) × P(Y=1)
Assumes conditional independence between sources given class label (Naïve Bayes assumption)
◈POSTERIOR PROBABILITY DISTRIBUTION
◈CALIBRATION — RELIABILITY DIAGRAM
Bayesian fusion produces the best-calibrated probabilities — closest to the diagonal.
